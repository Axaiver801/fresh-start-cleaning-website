# FreshStart Cleaning Website

A bilingual (English/Spanish) React landing page and business model for a sofa & rug cleaning service.

## Features
- English/Spanish toggle
- Pricing, team, and contact sections
- Booking form for quick requests

## Quick Start

1. **Clone the repo**
2. `npm install`
3. `npm start`
4. Visit [http://localhost:3000](http://localhost:3000)

## Tech
- React
- Tailwind CSS (via CDN for rapid prototyping)

## Deploy
- You can deploy with Vercel, Netlify, or GitHub Pages.
